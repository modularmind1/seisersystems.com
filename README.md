# Seiser Systems Website

Public-facing static website for Seiser Systems.

This repository is intentionally simple:

- HTML5
- CSS3
- Vanilla JavaScript when needed
- No frameworks
- No build tooling
- No backend

## Structure

- `index.html` - Current GitHub Pages entry point.
- `assets/` - Images, icons, and other static media.
- `css/` - Site stylesheets.
- `js/` - Future vanilla JavaScript modules.
- `pages/` - Future standalone public pages.
- `docs/` - Project documentation and architecture notes.
- `public/` - Reserved for a future public root if hosting changes.

See `docs/architecture-decisions.md` for structure decisions.
